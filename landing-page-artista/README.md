# Landing Page Artista

Este proyecto es una landing page diseñada para mostrar el trabajo de un artista. La página incluye una barra de navegación con enlaces a diferentes secciones, una hero section destacando una obra del artista y un botón de contacto a través de WhatsApp.

## Estructura del Proyecto

El proyecto contiene los siguientes archivos:

- **index.html**: Contiene la estructura semántica de la landing page, incluyendo una barra de navegación y secciones para Galería, Sobre mí, Trayectoria, Servicios, Eventos, Preguntas frecuentes y Contacto. La hero section presenta un título, una obra del artista y un botón de contacto por WhatsApp.

- **styles.css**: Define los estilos CSS para la landing page, asegurando un diseño responsivo y atractivo en diferentes dispositivos.

## Instrucciones de Configuración

1. Clona el repositorio en tu máquina local:
   ```
   git clone <URL_DEL_REPOSITORIO>
   ```

2. Navega al directorio del proyecto:
   ```
   cd landing-page-artista
   ```

3. Abre el archivo `index.html` en tu navegador para ver la landing page.

## Información Adicional

- Asegúrate de que todos los enlaces en la barra de navegación apunten a las secciones correctas.
- La página está diseñada para ser responsiva y seguir las mejores prácticas de SEO y GEO.
- Se ha implementado etiquetado semántico y Schema.org en la sección de Preguntas frecuentes para mejorar la visibilidad en motores de búsqueda.
